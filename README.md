# Magic Talk

Test voice communications with robots

# Requirements

* mpg321 (http://macappstore.org/mpg321/)

For Mac

```bash
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null
brew install mpg321
```

For Linux

```bash
sudo apt-get install mpg321
```
